## Story
As an analyst, I want to run the regression and immediately see fit quality and coefficients, so that I get the model applied automatically (no manual formulas).

## Acceptance Criteria (met)
- `POST /api/regress` runs OLS server-side and returns the full result.
- Summary cards: R2, adjusted R2, train/test row counts, degrees of freedom.
- Coefficients table: intercept + each active X with coefficient, std error, t, p-value, significance marker.
- Model warnings surfaced (collinearity, low degrees of freedom).

- `app/api/regress/route.ts`
- `components/Results.tsx` (cards, coefficients table, warnings)
- `lib/stats/*` (engine)
