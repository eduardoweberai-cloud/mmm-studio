## Story
As an analyst, I want to toggle variables, see multicollinearity diagnostics, and simulate Y from hypothetical inputs, so that I can explore scenarios safely.

## Acceptance Criteria (met)
- Variable activation toggles; model recomputes on the selected X only.
- VIF diagnostics table per variable with high (>5) flags.
- What-if simulator: enter hypothetical X values, get a live Y prediction (client-side from fitted coefficients).

## Implemented by
- `app/page.tsx` (variable toggles)
- `components/Results.tsx` (VIF table, simulator)
- `lib/stats/vif.ts`, `lib/stats/index.ts` (predictY)
