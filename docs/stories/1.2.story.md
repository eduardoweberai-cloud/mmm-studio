## Story
As an analyst, I want to enter monthly data in a grid, paste straight from Excel, or import a CSV, so that getting data into the model is effortless.

## Acceptance Criteria (met)
- Editable grid, default 36 rows, Y + up to 20 X columns; add/remove rows and X columns.
- Paste-from-Excel into any cell fills the grid (TSV/CSV auto-detected).
- CSV import with optional header row.
- Brazilian and US number formats accepted ("1.234,56", "1234,56", "1234.56").
- Configurable train/test split slider (50-90%), default 70% with live train/test row counts.
- "Carregar exemplo" loads a realistic 36-month dataset.

## Implemented by
- `app/page.tsx` (grid, toolbar, paste, controls)
- `lib/parse.ts` (delimited parsing + number coercion)
- `lib/sample.ts` (example dataset)
