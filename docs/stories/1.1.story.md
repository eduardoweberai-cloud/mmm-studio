### Algorithm spec (from architecture section 6, authoritative)
- Split: chronological, no shuffle. `trainCount = round(n * trainRatio)`. Train = first `trainCount` rows, test = remainder.
- Design matrix on training rows: `Xd = [1 | X_active]`.
- Fit: `beta = solve(Xd'Xd, Xd'y)`. If near-singular, use SVD pseudo-inverse and warn `"multicolinearidade: coeficientes podem ser instaveis"`.
- `SSE = e·e` with `e = y - Xd·beta`; `SST = sum((y - mean(y))^2)`; `R2 = 1 - SSE/SST`.
- `adjR2 = 1 - (1 - R2) * (n - 1) / (n - k - 1)`. If `n - k - 1 <= 0`, set adjR2 to null and warn low degrees of freedom.
- `sigma2 = SSE / (n - k - 1)`; `Var(beta) = sigma2 * (Xd'Xd)^-1`; `stdErr = sqrt(diag)`; `t = beta / stdErr`; `p = 2 * (1 - studentt.cdf(abs(t), df = n - k - 1))`.
- VIF for active X column j: regress `X_j` on the other active X + intercept, get `R2_j`, `VIF_j = 1 / (1 - R2_j)`. Flag `VIF > 5`, strong flag `VIF > 10`.
- Contribution percent: `contrib_j = sum_i abs(beta_j * X_ij)` over training rows; `pct_j = 100 * contrib_j / sum_j contrib_j`. Document in code that this is a practical signal-share decomposition, not a variance decomposition.
- Prediction (test + simulator): `yhat = intercept + sum_j beta_j * x_j`.
- Test metrics: `MAPE = mean(abs(a - p) / abs(a))` (skip rows where `a = 0`); `RMSE = sqrt(mean((a - p)^2))`; `testR2 = 1 - SSE_test / SST_test` (SST_test around the test-set mean).

### Library hints
- `ml-matrix`: `new Matrix(rows)`, `.transpose()`, `.mmul(other)`, `inverse(matrix)`, `solve(A, b)`, `pseudoInverse(matrix)` (or via SVD).
- `jstat`: `jStat.studentt.cdf(t, df)` for p-values.
- Singularity check: test the determinant or reciprocal condition number of `Xd'Xd`; below a small epsilon, take the pseudo-inverse path.

### Reference fixture generation (run once, offline, then commit the JSON)
Use a seeded dataset so the test is deterministic and self-contained at run time (no Python needed in CI):

```python
# scripts/gen_fixture.py  (run once locally; commit lib/stats/__tests__/fixture.json)
import numpy as np, json, statsmodels.api as sm
rng = np.random.default_rng(42)
n, k = 36, 3
X = rng.normal(size=(n, k))
beta_true = np.array([2.0, -1.0, 0.5])
y = 5 + X @ beta_true + rng.normal(scale=0.5, size=n)
Xd = sm.add_constant(X)
res = sm.OLS(y[:25], Xd[:25]).fit()  # train = first 25
json.dump({
  "x": X.tolist(), "y": y.tolist(), "trainRatio": 0.7,
  "expected": {
    "intercept": res.params[0], "beta": res.params[1:].tolist(),
    "r2": res.rsquared, "adjR2": res.rsquared_adj,
    "pValues": res.pvalues.tolist(), "stdErr": res.bse.tolist()
  }
}, open("lib/stats/__tests__/fixture.json", "w"), indent=2)
```
The golden test loads `fixture.json`, calls `fit()`, and asserts tolerances (beta and R2 within 1e-6, p-values within 1e-4).
